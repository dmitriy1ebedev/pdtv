# Changelog

## 1.1

### Исправлено
- **Ярлыки для нескольких дежурных.** Раньше все ярлыки записывались в один
  файл `pdtv.desktop` — созданный вторым дежурным ярлык затирал ярлык первого.
  Теперь у каждого дежурного свой файл (в имени и подписи ярлыка — его ФИО),
  ярлыки живут рядом и не мешают друг другу. Если тот же дежурный создаёт
  ярлык повторно — обновляется только его ярлык.
- **Ярлык кладётся на видимый рабочий стол.** На Astra ярлык теперь всегда
  попадает в `~/Desktops/Desktop1` — тот стол, который реально показан на
  экране. Раньше система могла подсунуть каталог `~/Desktop`, которого на
  экране нет, и ярлык «исчезал».

### Добавлено
- **Защита от двойного запуска.** Если обработка в этой рабочей папке уже
  идёт (например, второй дежурный кликнул свой ярлык), новый запуск честно
  сообщает об этом и не вмешивается — файлы не перемешиваются.
